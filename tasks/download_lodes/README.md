# Download LODES

Downloads 2019 LODES OD, RAC, and WAC files for Illinois and filters them to Cook County.

These commuting and employment files are used by gravity estimation and inversion.

## Running

```bash
cd code && make
```

Main outputs: `output/lodes_od_cook_county_2019.csv`, `output/lodes_rac_cook_county_2019.csv`, and `output/lodes_wac_cook_county_2019.csv`.
